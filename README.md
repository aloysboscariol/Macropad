### Inspiration

I wanted to create a macropad for my soundboards! I created a macropad using 6 switches, 1 rp2040, and 2 leds. 

### Challenges

That was my first time using Fusion 360, KiCad, and even my first real project for Hackclub. It took a lot of thought and research online to achieve this result. I also received help from many people on Slack.

### Specifications

BOM: 
- 6x Cherry MX Switches
- 2x SK6812 MINI Leds
- 1x XIAO RP2040
- 6x Blank DSA Keycaps
- 4x M3x16 Bolt
- 4x M3 Heatset

Others:
- KMK Firmware
